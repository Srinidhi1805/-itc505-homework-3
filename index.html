Exploring Custom Fonts: Curlz MT

My Font Adventure

In this page, I'll guide you through the process of serving a unique font like Curlz MT to the web browsers.

Instructions:

  1. Font Selection and Downloading

    Firstly Im choosing Curlz MT as the font and downloaded all the required font files like tff,woff and woff2 files.

  2. CSS Coding

    Edit your CSS file and define the @font-face rule to introduce your font to the website and give the correct path of the font files.I am also doing some custom font colors to just make sure the html looks cool!

  3. Setting up server

    Upload CSS file and other font files to your serve

  4. Sharing Your Experience

    Create a webpage like this one to share your font-serving journey with others, demonstrating how the font appears. Name this as index.html and upload to the server where your css and font files are.

  5. Running on the server

    Im using github pages to deploy this webpage and serve the font. So create a repo and add all your files to the repo and deploy the project repo to github pages
